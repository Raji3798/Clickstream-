# Clickstream Project 
# 🛒 Clickstream Data Analysis & Prediction System

## 📌 Project Overview

This project focuses on analyzing e-commerce clickstream data to understand customer behavior and build intelligent machine learning models. The goal is to enhance customer engagement, predict purchase intent, estimate revenue, and segment users for targeted marketing.

The application is built using **Python, Machine Learning, and Streamlit**, providing an interactive interface for real-time predictions.

---

## 🎯 Objectives

* 🔍 Predict whether a customer will complete a purchase (**Classification**)
* 💰 Estimate potential revenue generated by a customer (**Regression**)
* 👥 Segment users into behavioral groups (**Clustering**)
* 📊 Provide actionable insights for business decision-making

---

## 🧠 Business Use Cases

* **Customer Conversion Prediction** – Identify high-potential buyers
* **Revenue Forecasting** – Optimize pricing and marketing strategies
* **Customer Segmentation** – Enable personalized recommendations
* **Churn Reduction** – Detect drop-off behavior early
* **Improved Targeting** – Enhance campaign efficiency

---

## 📂 Dataset Description

The dataset contains clickstream data from an e-commerce platform, including:

* Session details
* Product categories
* Page visits
* User location
* Product attributes (color, price, etc.)

Key features used:

* MONTH, DAY
* COUNTRY
* PAGE
* PRODUCT CATEGORY
* COLOUR
* LOCATION
* MODEL PHOTOGRAPHY

## ⚠️ Challenges Faced

* Handling semicolon-separated dataset
* Data leakage due to price-related features
* Feature mismatch between training and UI
* Model overfitting (initially 100% accuracy)

---

## ✅ Solutions Implemented

* Used `sep=';'` to correctly load dataset
* Removed leakage features (PRICE, SESSION ID, etc.)
* Standardized feature pipeline
* Ensured consistent preprocessing in Streamlit

---

## 🧠 Key Learnings

* Importance of avoiding data leakage
* Feature selection impacts model performance
* Real-world ML requires proper validation
* Deployment must match training pipeline
